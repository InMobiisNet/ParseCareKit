Note

open class Note : PCKObjectable

An Note is the ParseCareKit equivalent of OCKNote. OCKNote can be attached to all other CareKit objects and values. Use cases may include a physician leaving a note on a task when it is modified to explain why a medication dose was changed, or a note left from a patient to a care provider explaining why they weren’t able to complete a task on a certain occasion.
