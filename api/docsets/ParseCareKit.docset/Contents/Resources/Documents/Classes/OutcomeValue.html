OutcomeValue

open class OutcomeValue : PCKObjectable

An OutcomeValue is the ParseCareKit equivalent of OCKOutcomeValue. An OCKOutcomeValue is a representation of any response of measurement that a user gives in response to a task. The underlying type could be any of a number of types including integers, booleans, dates, text, and binary data, among others.
